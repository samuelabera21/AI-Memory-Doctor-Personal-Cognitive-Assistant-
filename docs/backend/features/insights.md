# Insights and Behavior Analysis

## Goal
Detect behavioral patterns and provide reflective insights.

## Endpoint
- GET /insights

## Analysis
- Most common memory type
- Most productive time bucket
- Repeated mistake themes
- Trend direction (increasing, decreasing, stable)

## Output
Structured payload including narrative and key analytics fields.
